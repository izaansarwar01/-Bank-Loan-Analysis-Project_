   # 🏦 Bank Loan Analysis Project

## 📌 Project Overview

This project analyzes financial loan data to evaluate lending performance, borrower risk, and overall portfolio health using Python.

The objective is to extract meaningful business insights from raw loan data and identify performance trends, default risk levels, and funding distribution patterns.

---

## 📊 Key KPIs Analyzed

- Total Loan Applications  
- Month-To-Date (MTD) Applications  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average Debt-to-Income Ratio (DTI)  
- Good Loan vs Bad Loan Analysis  

---

## 📈 Analysis Performed

- Monthly Trend Analysis  
- Regional (State-wise) Funding Analysis  
- Loan Term Distribution  
- Employment Length Analysis  
- Loan Purpose Breakdown  
- Home Ownership Impact on Funding  

---

## 🛠 Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Plotly  

---

## 🚀 Business Insights

- Evaluated loan portfolio health using Good vs Bad loan segmentation  
- Identified high-performing states and funding concentration  
- Analyzed borrower risk exposure using DTI and interest rate metrics  
- Assessed repayment efficiency through funded vs received comparison
